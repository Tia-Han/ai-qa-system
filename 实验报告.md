## 一、实验内容

开发一个简单的人工智能基础问答系统，支持用户输入问题，系统根据预设的知识库进行匹配并返回答案。具体要求：

1. **构建知识库**：预设20个人工智能基础相关问题及对应答案，以字典形式存储；提取所有问题的核心关键词存入集合；按关键词分类优化查询效率。
2. **用户交互**：提供可视化交互界面，支持多次提问，直到用户输入"退出"为止，将用户所有提问记录存入列表。
3. **问答匹配**：实现关键词匹配逻辑（提取用户问题中的关键词，转换为集合，与知识库问题的核心关键词集合求交集，交集元素最多的问题即为匹配度最高的问题），返回对应答案；若未匹配到，返回提示信息。
4. **版本管理**：采用Git进行版本管理，并将最终系统提交到GitHub。

```python
def extract_keywords(text):
    stop_words = {'什', '么', '是', '的', '在', '中', '和', '与', ...}
    keywords = set()
    for char in text:
        if '\u4e00' <= char <= '\u9fa5':
            if char not in stop_words:
                keywords.add(char)
    english_words = re.findall(r'[a-zA-Z]+', text)
    for word in english_words:
        if len(word) >= 2:
            keywords.add(word.lower())
    return keywords
```

**问答匹配函数**：
```python
def match_question(user_question):
    user_keywords = extract_keywords(user_question)
    candidate_questions = set()
    for keyword in user_keywords:
        if keyword in keywords_to_questions:
            candidate_questions.update(keywords_to_questions[keyword])
    
    max_match_count = 0
    best_match = None
    for question in candidate_questions:
        question_keywords = extract_keywords(question)
        intersection = user_keywords & question_keywords
        match_count = len(intersection)
        if match_count > max_match_count:
            max_match_count = match_count
            best_match = question
    
    return best_match, knowledge_base.get(best_match, "未找到答案"), max_match_count
```

## 四、结果分析

### 4.1 遇到的问题及解决方案

**问题1：关键词提取不准确**
- **现象**：正则表达式把整个句子作为一个词提取，导致匹配效果不佳。
- **解决方案**：修改为按单个汉字提取关键词，同时支持英文单词提取，增加停用词过滤。

**问题2：非监督学习匹配错误**
- **现象**："非监督学习有哪些方法"匹配到了"什么是监督学习？"而不是"什么是非监督学习？"。
- **解决方案**：将"非"字从停用词中移除，确保"非监督学习"和"监督学习"能正确区分。

**问题3：匹配逻辑效率低**
- **现象**：原匹配逻辑遍历所有知识库问题，效率较低。
- **解决方案**：利用`keywords_to_questions`分类结构，先通过关键词定位候选问题，再进行精确匹配。
