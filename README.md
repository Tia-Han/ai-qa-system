# 人工智能基础问答系统

一个基于Python的人工智能基础问答系统，支持关键词匹配、可视化交互界面和单元测试。

## 📁 项目结构

```
├── ai_qa_system.py          # 主程序（含Tkinter可视化界面）
├── web_app.py               # Web版本（Flask）
├── test_ai_qa_system.py     # 单元测试文件（23个测试用例）
├── requirements.txt         # Python依赖
├── 实验报告.md              # 实验报告
├── push_to_github.py        # GitHub推送辅助脚本
└── README.md                # 项目说明文档
```

## ✨ 功能特性

- 📚 **知识库**：包含20个人工智能基础问题及答案
- 🔍 **关键词匹配**：基于关键词集合交集的智能匹配算法
- 🖥️ **可视化界面**：使用Tkinter构建美观的GUI界面
- 🌐 **Web版本**：使用Flask构建的Web应用，支持在线访问
- 🧪 **单元测试**：23个测试用例，确保代码质量
- 📊 **历史记录**：记录用户所有提问

## 🚀 快速开始

### 环境要求

- Python 3.11+
- Flask 2.0+（Web版本）
- Tkinter（Python内置，无需额外安装）

### 安装依赖

```bash
pip install flask
```

### 运行方式

#### 方式1：运行Web版本（推荐，支持在线访问）

```bash
python3 web_app.py
```

然后在浏览器中打开：
- 本地：http://127.0.0.1:8080
- 局域网：http://your-ip:8080

#### 方式2：运行GUI界面

```bash
python3 ai_qa_system.py
```

#### 方式3：运行命令行版本

```bash
python3 -c "from ai_qa_system import main_cli; main_cli()"
```

#### 方式4：运行单元测试

```bash
python3 -m unittest test_ai_qa_system.py -v
```

## 📖 使用说明

### Web版本使用方法

1. **启动服务**：运行 `python3 web_app.py` 启动Web服务
2. **打开浏览器**：访问 http://127.0.0.1:8080
3. **选择问题**：从左侧知识库列表中点击问题，或在输入框中输入您的问题
4. **查看回答**：点击"提问"按钮或按回车键，系统会显示匹配结果和回答

### GUI版本使用方法

1. **启动程序**：运行 `python3 ai_qa_system.py` 启动GUI界面
2. **输入问题**：在输入框中输入您的问题，按回车键或点击"提问"按钮
3. **查看回答**：系统会自动匹配知识库中的问题并显示答案
4. **查看历史**：切换到"提问记录"标签页查看所有提问
5. **清空记录**：点击"清空"按钮清空所有记录
6. **退出程序**：点击"退出"按钮关闭程序
